# 브리핑 이력 로그 (회계팀 오전 브리핑 루틴 전용)

이 파일은 "회계팀 오전 브리핑" 루틴이 실행될 때마다 자동으로 읽고 갱신하는 이력 로그입니다.
루틴은 매일 새로운 세션에서 실행되어 이전 대화 기록을 전혀 기억하지 못하므로,
전날 이전에 어떤 카드·주제를 다뤘는지는 이 파일을 통해서만 확인할 수 있습니다.

## 사용 규칙

- 브리핑 작성 시작 전, 이 파일의 최근 항목(최대 5영업일치)을 먼저 읽고 중복 여부를 판단한다.
- 브리핑 완료 후, 아래 "항목 형식"에 맞춰 오늘 날짜 항목을 이 파일 맨 위(사용 규칙 다음, 이력 목록 맨 앞)에 추가한다.
- 이력은 항상 최근 5영업일치만 남기고, 그보다 오래된 항목은 삭제한다.
- 공개 출처 기반 요약만 기록한다. 원문 HTML 전체, 비공개 재무자료, 개인정보는 저장하지 않는다.
- 사용자가 "테스트" 실행이라고 명시한 경우, 이 파일은 갱신하지 않는다.

## 항목 형식

```
## YYYY-MM-DD
- 섹션명: 카드 제목 또는 주제 | 핵심 키워드 3~6개 | 주요 출처명 | 전일 대비 새로웠던 점
```

섹션당 여러 카드가 있으면 카드마다 한 줄씩 기록한다.
